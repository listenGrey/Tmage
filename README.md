# Tmage
> If you want to read English file, please click *[here]("https://github.com/listenGrey/Tmage/blob/dev/others/enREADME.md")*.

### 简介
Tmage是一个基于Web的，分布式图片存储与分享平台，用户可以上传图片，并可以根据标签来快速的搜索出想要的图片。

---

### 技术栈
- 前端：VUE3
- 后端：Gin
- 数据库：MongoDB、Redis、MySQL
- 消息队列：kafka
- 数据传递：gRpc
- 文件存储：分布式存储系统

### 主要功能
- 用户注册和登录
- 图片上传
- 图片下载
- 删除图片
- 编辑图片
- 搜索图片
- 存储容量管理
